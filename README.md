Welcome to the GitHub repository of Rhythmix, a cutting-edge Music Recommendation System, and RhythmoBot, an NLP-powered music recommendation chatbot!

## Description

Rhythmix is a powerful music recommendation system that leverages state-of-the-art machine learning algorithms and collaborative filtering techniques to curate a delightful music experience for users. With a vast database of songs and a sophisticated recommendation engine, Rhythmix analyzes user interactions and behavior to understand their unique taste in music. This results in highly accurate and relevant song recommendations, ensuring users discover new tracks they will love.

RhythmoBot, on the other hand, is an intelligent chatbot integrated with Rhythmix. Powered by natural language processing (NLP) algorithms, RhythmoBot can engage in natural and interactive conversations with users to understand their music preferences. Users can simply chat with RhythmoBot, share their mood, favorite artists, or any specific song requirements, and RhythmoBot will instantly suggest personalized music recommendations based on the input.

## Key Features

- **Personalized Recommendations:** Rhythmix tailors recommendations for each user, ensuring they receive songs that match their individual preferences.
- **Genre and Mood-based Filtering:** Users can explore music based on specific genres, moods, or activity types, catering to their current preferences.
- **Seamless Integration:** Rhythmix can be seamlessly integrated into music streaming platforms, apps, or websites to enhance the user experience.
- **Real-time Updates:** Rhythmix regularly updates its recommendation models to provide the most up-to-date song suggestions.
- **Discover Weekly Playlists:** Users receive weekly playlists with fresh recommendations based on their listening history and trending tracks.
- **NLP-Powered RhythmoBot:** Engage in natural conversations with RhythmoBot to receive personalized music recommendations effortlessly.

## Technologies Used

Rhythmix and RhythmoBot utilize a variety of cutting-edge technologies to deliver top-notch music recommendations:
- **Python:** The core programming language used for developing the recommendation algorithms and chatbot logic.
- **Machine Learning Libraries:** Rhythmix utilizes popular ML libraries like TensorFlow, Scikit-learn, and Keras for building and training recommendation models.
- **Collaborative Filtering:** Leveraging collaborative filtering techniques, Rhythmix analyzes user behavior and interaction patterns to generate relevant recommendations.
- **Natural Language Processing (NLP):** RhythmoBot uses NLP algorithms and frameworks like NLTK or spaCy to process user input and provide interactive responses.

For detailed installation instructions and usage guidelines, please refer to the repository's Wiki section. Feel free to explore the code, fork the project, and submit pull requests to make Rhythmix and RhythmoBot even better!



## Initial Setup:
Clone repo and create a virtual environment
```
$ git clone
$ cd Rythmix
```
Install dependencies
```
$ pip install Flask torch torchvision nltk
```
Install nltk package
```
$  python
>>> import nltk
>>> nltk.download('punkt')
```
Modify `intents.json` with different intents and responses for your Chatbot

Run
```
$ python train.py
```
This will dump data.pth file. And then run
the following command to test it in the console.
```
$  python chat.py
```
